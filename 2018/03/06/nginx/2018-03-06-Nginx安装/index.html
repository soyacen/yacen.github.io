Nginx安装

Nginx安装有好几种方式：

Debian/Ubuntu

apt install nginx

Red/CentOS

yum install nginx

源码安装

  1. 从官网下载源码
    最好下 stable 稳定版本wget https://nginx.org/download/nginx-1.4.7.tar.gz

  2. 解压
    tar -xzf nginx-1.4.7.tar.gz && cd nginx-1.4.7

    nginx源码目录是这样：

    auto CHANGES CHANGES.ru conf configure contrib html LICENSE man README src
    CHANGES CHANGES.ru 是nginx英文俄文写的版本历史
    conf 是nginx的配置文件目录
    html 是nginx默认的页面
    man 是nginx手册
    src 就是源码
    configure和auto 是编译nginx所用到的文件

  3. 配置
    ./configure --prefix=/usr/local/nginx
    中间可能需要安装 zlib pcre 等库

  4. 安装
    make && make install && cd /usr/local/nginx
    nginx比较小，编译的很快
  5. 运行
    ./sbin/nginx
    curl 'localhost'
    如果能请求成公，则代表安装成功了
