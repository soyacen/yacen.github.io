Redis安装与使用

redis安装

redis有几种安装方法：

  1. apt/yum install redis
  2. 源码安装

源码安装

  1. 官网下载压缩包
  2. tar -xzf XXX.tar.gz
  3. cd redis_dir
  4. make PREFIX=/usr/local/redis && make install
  5. cp *.conf /usr/local/redis/conf

启动服务

/usr/local/redis/bin/redis-server /usr/local/redis/conf/redis.conf

连接

/usr/local/redis/bin/redis-cli [-h localhost -p 6379 ]

让redis以后台进程的形式运行

编辑conf配置文件,修改如下内容:

daemonize yes
