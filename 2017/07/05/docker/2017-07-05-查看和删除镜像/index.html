查看和删除镜像

查看和删除镜像

Docker Image 镜像

  • 容器的基石
  • 层叠的只读文件系统
  • 联合加载（union mount）

镜像的存储位置

也可以通过docker info 查看信息

列出镜像

  • -a, –all=false： 所有镜像，默认不显示中间层镜像
  • -f, –filter=[]：过滤条件
  • –no-trunc=false：不使用截断的形式显示数据
  • -q, –quiet=false：只显示镜像id

镜像的仓库

  • REPOSITORY 仓库
    镜像的集合，相关联的镜像
  • REGISTRY 仓库？

镜像的标签

  • TAG
    • ubunut:14.04
    • centos:latest

查看镜像的详细信息

  • -f, –format=””

删除镜像

  • -f, –force=false: Force removal of the image,强制删除镜像
  • –no-prune=false: Do not delete untagged parents，保留未打标签的父镜像
